css学习笔记(一)

  • 前端
  • 2017-02-19

一、 三种样式的使用范围

样式名称 存放位置 作用范围 应用
嵌套样式 head中的title 下面 只能作用在当前标签之中 存放一些当前页面会用到的样式
行内样式 标签中通过style 来设置 当前样式所处的页面 一般情况用的少（整个页面只有这个标签用了这个样式，才会用）
外联样式 在其他文件中，通过link标签 所有的页面都可以使用 存放一些所有页面都会用到的公共样式

二、 属性学习

  1. text-index

    作用：设置首行缩进。

    用法: text-index: 2em;

    em: 一个文本的距离

2.text-align:

作用：设置文本的位置


  取值：

   center: 设置内容在容器的水平方向居中。

   left: 设置内容在容器的水平方向靠左。

   right: 设置内容在容器的水平方向靠右。

3.text-decotion

作用： 设置文本的装饰

取值：

   none: 没有任何装饰

   underline: 下划线

4.margin: 0 auto;

作用： 设置容器自身水平居中。

5, css 三大特性

继承性

   子元素可以继承父元素的样式。

   以text－，font－，line－开头的属性可以继承
   color 可以继承

   1. a 标签的颜色不能继承，需要改a标签的颜色，直接 作用在a标签上。
   2. h 标签字体的大小不能被继承，需要改h字体大小，直接 作用在h标签上。
   3. div 高度如果不设置由内容决定（没内容高度为0),宽度默认由氟元素继承过来。

层叠性

优先级
！important>行内样式>id选择器 >类选择器>通配符>浏览器默认样式>继承


!important属性无法继承。

  1. css 权重

    作用：多个选择器组合以后的优先级
    算法：（0，0，0，0）

    第一个0对应的是important 的个数
第二个0对应的是id选择器的个数
第三个0对应类选择器的个数
第四个0对应标签选择器的个数

    比较：

    先从第一个0开始比较，如果第一个0大，
那么说明这个选择器的权重高，如果第一个相同，
比较第二个，依次类推。

    选择器在查找元素的时候不是从左往右找，而是从右往左找。
